# My Full Stack Notes App

## Backend

- Framework: Spring Boot v3.2.1
- Maven: v4.0
- JDK: Java 17

### Run Backend

cd notes-backend/notes
./mvnw spring-boot:run

## Frontend

- Framework: React v18.2.0
- Node.js: v20.10.0

### Run Frontend

cd notes-frontend
npm install
npm run dev

### Database

-MySQL Server: v8.0

### App Run

./setup.sh
